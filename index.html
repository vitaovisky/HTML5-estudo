header é a tag que engloba todo o cabeçalho da página

  • Junto ao header é normal ter a tag nav, junto de listas para organizar o menu de navegação

  • Separando os itens da navegação em listas

  • Podendo se utilizar de ul e ol para listas não-ordenadas e listas ordenadas respectivamente

  • E li para os itens das listas.

main é a tag que engloba todo o corpo da página

h1 até h6 são as tags de título do site, em ordem de importância do 1 ao 6

A tag img agrega imagens a página, é importante ter um atributo alt nessas tags para a acessibilidade. Como forma semântica, caso outras mídias sejam adicionadas, há tags específicas para elas, como audio, video.

p é a tag que irá gerar parágrafos no html

As tags, i itálico, u underline, mark marca-texto

div são tags somente estruturais, não possuem peso semântico

section também terão utilização similar a div, porém com peso semântico, marca seções de conteudo

article também terão utilização similar a div, porém com peso semântico, article normalmente será utilizado para um contaudo independente do contexto da página

A tag a irá ancorar algum link externo ou parte do projeto web. Por padrão irá abrir o link na mesma aba, para abrir em aba diferente é necessário o atributo target:_blank

Para ancoragem do link da mesma página ou documento é utilizado o atributo id como demonstrado no próximo link ->Voltar ao topo

A tag form irá agrupar um formulário, nela a tag caption cria uma legenda do formulário e o atributo summary auxiliar na acessibilidade

O atributo method do form irá gerar a forma de envio do formulário ,sendo post a forma segura, o metodo get envia por http.








São botões dentro do formulário com funções específicas para o formulário. Apesar de que a tag button poder ser utilizada fora do formulário.

O atributo type designa o que estes botões irão realizar, submit(enviar o formulário) ou reset(limpar os campos)

Iframe é uma tag para incorporar informações externas ao site. Irá rodar outro documento dentro da página atual.

footer é a tag que engloba todo o rodapé da página.

Normalmente são adicionados créditos, informações de contato, uma navegação secundária, etc.
